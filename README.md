# cusec-mc
Configuration for a CUSEC Minecraft Server!


## compose.yaml
Use this tool [here](https://setupmc.com/java-server/) to help create the compose.yaml!

## configuation
See [the Docker Minecraft Server](https://docker-minecraft-server.readthedocs.io/en/latest/) for more deets

Merges into main will automatically update the server. please ensure that the server is not in use before merging.
